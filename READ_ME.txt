Here's some info for the bot

1) Run the jar. It'll generate a settings.txt. Add your token then run it again and it should work
2) To run it through Intellij, use the task "run"
3) to save it as a jar, run the shadowJar task
4) to run the jar, open up cmd, go to the directory, and then type "java -jar [jar].jar"
5) or run the BAT. Change the directory to the jar file's directory
